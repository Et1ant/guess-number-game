# Guess the Number Game

Консольная реализация игры «Угадай число» на языке Python. Проект демонстрирует применение базовых алгоритмов, работу с файловой системой и обработку пользовательских исключений.

## Функциональные возможности

* **Динамическая сложность:** Выбор одного из трех уровней (Easy, Medium, Hard), влияющих на количество доступных попыток.
* **Система рекордов:** Автоматическое сохранение и чтение лучшего результата (минимальное число попыток для победы) через локальный файл.
* **Отказоустойчивость:** Обработка некорректного ввода данных (строки, спецсимволы) с помощью блоков `try-except`, что предотвращает аварийное завершение программы.
* **Обратная связь:** Интерактивные подсказки («Too low», «Too high») на основе сравнения пользовательского ввода с загаданным числом.

## Технические особенности

### Работа с файлами и данными

Для управления ресурсами используется менеджер контекста `with open()`. Программа считывает данные из `record.txt` при запуске. Если файл отсутствует, начальное значение рекорда устанавливается как `float('inf')` для обеспечения корректной логики сравнения при первой победе.

### Управление потоком (Control Flow)

Логика проверки количества попыток реализована с использованием конструкции `for...else`. Блок `else` выполняется только при полной итерации цикла без вызова `break`, что позволяет эффективно разделить сценарии победы и проигрыша без использования вспомогательных переменных-флагов.

## Структура проекта

* `main.py` — основной файл с исходным кодом игры.
* `record.txt` — локальное хранилище для текущего рекорда.
* `.gitignore` — файл конфигурации Git для исключения локальных данных из индекса репозитория.
* `README.md` — документация проекта.
